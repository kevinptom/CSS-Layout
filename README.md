A clean, responsive webpage layout built to demonstrate a practical understanding of fundamental CSS layout techniques, including Flexbox, CSS Grid, and structural positioning.

## 🎯 Project Objective
This project was developed as part of a Web Fundamentals course task. The primary goal was to step away from CSS frameworks and build a robust, classic webpage architecture entirely from scratch using modern vanilla CSS building blocks.

## ✨ Layout Techniques Utilized
* **Flexbox Container (`display: flex`):** Used on the main wrapper to seamlessly align the sidebar and main content areas, as well as to create a "sticky footer" that always stays at the bottom of the viewport.
* **CSS Grid (`display: grid`):** Implemented in the main content area to create a responsive, auto-wrapping card layout without needing complex media queries.
* **Sticky Positioning (`position: sticky`):** Applied to the header element to keep navigation context visible while the user scrolls down the page.
* **Responsive Design:** Utilizes a mobile-first approach with media queries (`@media`) to transition from a stacked layout on mobile devices to a dual-column layout on larger screens.
